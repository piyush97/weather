# Weather Information App By Piyush Mehta

## File structure

```structure
|-- node_modules
|-- weather-app
   |-- views
      |-- index.ejs
   |-- public
      |-- css
         |-- style.css
   |-- package.json
   |-- server.js
```

## Technologies used
1. NodeJS

## Api used
1. openweathermap

## How to deploy this app?

```bash
$ npm install npm@latest -g
$ node -v
$ git clone https://github.com/piyush97/weather.git
$ cd weather
$ node server.js
// Weather App is running on port 3000
```
